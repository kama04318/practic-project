# КДЛ Медик



###### Автор

Халимов Камиль— студент группы ИСП3323



###### Описание проекта

Веб-приложение для клинико-диагностической лаборатории КДЛ Медик.

Разработано на ASP.NET Core (C#).



###### Функционал

\- Просмотр списка анализов с ценами и сроками готовности

\- Запись пациента на анализ через онлайн-форму



###### Технологии

\- C# / ASP.NET Core Razor Pages

\- HTML / CSS
